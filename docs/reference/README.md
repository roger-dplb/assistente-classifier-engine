# 📖 Referência (Reference)

Documentação técnica precisa para consulta rápida.

## O que é Referência?

Referência é **informação factual** organizada para consulta:
- 🔍 **Busca rápida** - encontre o que precisa em segundos
- ✅ **Precisão** - fatos verificáveis e atualizados
- 📋 **Organização** - estruturada para navegação fácil
- 🚫 **Sem narrativa** - apenas os fatos

> 💡 **Use quando:** Você sabe o que procura e precisa dos detalhes técnicos.
