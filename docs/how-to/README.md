# 🛠️ Como Fazer (How-To Guides)

Guias práticos para resolver problemas específicos.

## O que são How-To Guides?

Diferente dos tutoriais, os How-To Guides:
- 🎯 **Focam em resolver um problema** específico
- ⚡ **São diretos** - sem contexto desnecessário
- 🔧 **Assumem conhecimento prévio** do sistema
- 📋 **Mostram a solução** em passos claros

```markdown

## 🎯 Problema

## 📋 Pré-requisitos
- Requisito 1
- Requisito 2
- Link para documentação relacionada

## 🚀 Solução

### Passo 1: [Ação]
```comando ou código```
Explicação breve

### Passo 2: [Ação]
...

## ✅ Verificação
Como confirmar que funcionou

## ❌ Resolução de Problemas
Erros comuns e suas soluções

## 📚 Veja Também
- [Link para guia relacionado]
- [Link para explicação conceitual]
```
